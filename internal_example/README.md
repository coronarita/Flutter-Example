# internal_example

공유환경설정에 카운트값을 저장하는 앱
공유환경설정 파일은 내부 저장소에 파일 형태로 데이터를 저장하는 것이지만, 앱에서만 접근할 수 있는 특수한 목적의 데이터 저장소 임.
